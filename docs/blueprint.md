# **App Name**: LogiTrack

## Core Features:

- Fast Order Entry: Single-screen order entry with minimal fields and keyboard shortcuts for quick input.
- Automated WhatsApp Notifications: Automatically send WhatsApp notifications to customers for order received, out for delivery, and delivered status updates.
- Payment Method Selection: Allow users to select the payment method (Pix, Dinheiro, Cart√£o, Boleto/Link) during order entry and track payments.
- Real-Time Tracking: Generative AI powered tool, will provide live predictive information and status of order, considering traffic data, weather patterns and other events on the road.
- Order Management: List and filter orders by phone number, tracking code, name, status, and period.
- Reporting and Analytics: Generate reports on deliveries by status, period, driver, and financial data by payment method and period. Export CSV/PDF.
- User Authentication and Authorization: Secure the application with user authentication and role-based authorization (admin, operator, driver).

## Style Guidelines:

- Primary color: Deep blue (#1A237E) for trust and reliability.
- Background color: Very light blue (#E3F2FD) to ensure a clean, readable layout in both light and dark modes.
- Accent color: Vibrant orange (#FF9800) for calls to action and important notifications, providing high contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable user interface.
- Note: currently only Google Fonts are supported.
- Use clear and consistent icons to represent order status, payment methods, and other actions.
- Mobile-first, responsive layout with large buttons and high contrast elements for ease of use on all devices.
- Subtle animations for status changes and notifications to provide visual feedback without being distracting.
